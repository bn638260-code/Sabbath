SabbathCue supports three STT engines, and you can swap between them at
runtime from the settings dialog. They map to two operating modes:
zero-marginal-cost local recognition, or cloud streaming when you want a
managed STT service.

## Option 1: Vosk

Vosk is the default local speech recognition engine. Audio stays on the
operator machine and the worker emits transcript events into the same detection
pipeline as the cloud providers.

To install the local model for development:

```bash
bun run download:vosk
```

Release builds can bundle a self-contained worker:

```bash
bun run build:vosk-sidecar
```

The app checks the Vosk model, worker, and runtime from Settings -> Speech.

## Option 2: Deepgram

Deepgram is a cloud speech-to-text service that streams audio over a WebSocket.
SabbathCue also has a REST fallback that uploads short windows if the WebSocket
disconnects.

1. Get an API key at [deepgram.com](https://deepgram.com/).
2. Open Settings -> Speech.
3. Choose **Deepgram** and save the key.

The key is stored in the OS keychain.

## Option 3: Gladia

Gladia is another cloud STT option using the Gladia live API.

1. Get an API key at [gladia.io](https://www.gladia.io/).
2. Open Settings -> Speech.
3. Choose **Gladia** and save the key.

The key is stored in the OS keychain.

## Comparing Providers

|                     | Vosk (local)                       | Deepgram (cloud)          | Gladia (cloud)      |
| ------------------- | ---------------------------------- | ------------------------- | ------------------- |
| API cost            | Free                               | Per-minute billing        | Per-minute billing  |
| Internet required   | No                                 | Yes                       | Yes                 |
| Recognition mode    | Local worker + constrained grammar | Keyword-boosted streaming | Live streaming      |
| Transcript coverage | Best for verse/reference cues      | Full general speech       | Full general speech |
| Works offline       | Yes                                | No                        | No                  |

Pick Vosk for the no-cost offline path. Pick Deepgram or Gladia when you have
a reliable network and want cloud transcription quality.
